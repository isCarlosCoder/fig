# Declaração e atribuição

## Sintaxe de declaração simples, leitura natural.

```js
let x = 42
let name = "Carlos"
let flag = true
```

## Declaração sem inicialização

```js
let y
```

## Reatribuição e inicialização

```js
x = 100
name = "Nova string"
flag = false

y = true
```

## Regras

- `let` é obrigatório para declarar novas variaveis
- `=` sempre usado para inicializar ou reatribuir
- Não precisa declarar tipo explicitamente -- inferencia pelo valor
